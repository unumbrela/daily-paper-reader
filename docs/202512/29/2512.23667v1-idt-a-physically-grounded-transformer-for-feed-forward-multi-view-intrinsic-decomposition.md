### 1. 核心问题与整体含义

*   **研究背景**：内蕴分解旨在将图像分解为材质属性和光照成分，这对于重打光、材质编辑、3D重建和世界建模等任务至关重要。虽然基于扩散模型的单视角内蕴分解方法取得了显著进展，但在多视角设置下，这些方法通常独立处理每张图像，导致严重的视角不一致性问题。
*   **核心问题**：
    *   **视角一致性**：现有的单视角方法（尤其是扩散模型）在处理多视角数据时，难以保持材质外观和光照的一致性，特别是在处理镜面高光等视角依赖效应时。
    *   **效率限制**：迭代生成的扩散模型计算成本高，推理速度慢，且缺乏跨视角联合推理的机制。
*   **整体含义**：本文提出了 IDT（Intrinsic Decomposition Transformer），一个基于前馈 Transformer 的框架，旨在通过联合推理多视角输入，一次性生成视角一致的内蕴分解结果，解决多视角场景下的材质与光照解耦问题。

### 2. 提出的方法论

*   **核心思想**：
    *   采用**前馈框架**替代迭代生成，利用 Transformer 的注意力机制在单次前向传递中联合处理多视角图像。
    *   引入**基于物理的成像模型**，显式地将图像分解为漫反射反照率、漫反射着色和镜面着色，以实现朗伯与非朗伯光传输的分离。
*   **成像模型（公式化）**：
    *   对于视角 $v$，观测图像 $I_v$ 建模为：
        $$I_v(x) = A(x) \odot S^{\text{diff}}_v(x) + S^{\text{spec}}_v(x)$$
    *   其中：
        *   $A$：视角不变的漫反射反照率。
        *   $S^{\text{diff}}_v$：视角依赖的漫反射着色，捕获朗伯光照效应。
        *   $S^{\text{spec}}_v$：视角依赖的镜面着色，捕获非朗伯效应（如高光）。
    *   该模型将镜面反射作为加性分量独立处理，防止视角依赖的外观泄露到材质属性中。
*   **网络架构流程**：
    1.  **多视角 Transformer 编码器**：类似 VGGT 的架构，处理多视角输入图像，生成共享的潜在 Token 集合 $Z$。
    2.  **几何-外观 Token 专业化**：不显式划分 Token，而是通过特定任务的监督信号，使 Token 隐式地分化为几何 Token ($Z_{\text{geo}}$) 和外观 Token ($Z_{\text{app}}$)。
    3.  **外观适配器**：引入针对不同内蕴因子的特定适配器（Adapter），通过交叉注意力机制从共享 Token 中提取特定信息。
        *   查询来自场景级 Token（相机/寄存器 Token 的池化）。
        *   键和值来自所有视角的 Patch Token。
    4.  **光照表示**：引入紧凑的场景级光照表示，参数化为球面高斯混合，作为着色预测的条件信号。
    5.  **预测头**：基于适配后的特征，分别预测共享的反照率 $A$、共享光照 $L$ 以及各视角的漫反射/镜面着色。

### 6. 主要结论与发现

*   IDT 能够在单次前向传递中实现**多视角一致的内蕴分解**，无需迭代采样。
*   相比先前的内蕴分解方法，IDT 生成了更干净的漫反射反照率、更连贯的漫反射着色以及隔离度更好的镜面成分。
*   显式分离漫反射和镜面分量（物理接地模型）有效解决了视角依赖效应导致的材质不一致问题。
*   联合推理机制使得模型能够利用跨视角信息，显著提升了多视角场景下的分解质量。

### 7. 优点

*   **效率高**：前馈架构避免了扩散模型的迭代采样，推理速度更快，更适合实际应用。
*   **视角一致性强**：通过 Transformer 联合处理多视角输入，从根本上解决了单视角独立处理带来的不一致问题。
*   **物理可解释性**：采用了符合物理渲染的成像模型（漫反射+镜面分离），不仅结果更可控，还支持重打光等下游任务。
*   **架构设计巧妙**：利用“涌现”的 Token 专业化和外观适配器，在不硬编码分割的情况下，实现了几何与外观信息的有效解耦和路由。
