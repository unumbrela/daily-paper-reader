### 2. 提出的方法论
该论文提出了一个名为 **SRFA (Scalable Residual Feature Aggregation)** 的端到端框架，其核心流程包含以下五个关键阶段：

*   **预处理**：
    *   采用 CLAHE（限制对比度自适应直方图均衡化）增强局部对比度；
    *   使用高斯模糊减少高频噪声；
    *   应用 3×3 中值滤波去除椒盐噪声；

*   **分割 (MAGRes-UNet)**：
    *   使用 **MAGRes-UNet**（多注意力门控残差 U-Net）进行胰腺及肿瘤区域的分割。
    *   引入注意力门机制抑制无关背景，利用残差块增强特征传播并缓解梯度退化，融合多级特征以精确勾勒边界。
*   **特征提取**：
    *   使用 **DenseNet-121** 结合 **残差特征存储**。
    *   利用 DenseNet 的密集连接保留细粒度空间细节，同时通过 RFS 暂存不同深度的中间特征，实现低级纹理与高级语义的同时聚合，形成丰富的高维特征集。
*   **混合元启发式特征选择**：
    *   结合 **哈里斯鹰优化 (HHO)** 和 **蝙蝠算法 (BA)**。
    *   HHO 模拟协同狩猎策略（探索与开发），BA 利用回声定位原理（基于频率、响度和脉冲率优化位置）。两者融合旨在从高维特征空间中筛选出最具判别力的特征子集。
*   **混合分类与优化**：
    *   构建由 **Vision Transformer (ViT)** 和 **EfficientNet-B3** 组成的混合分类器。ViT 捕捉全局上下文和长距离依赖，EfficientNet-B3 提取高效的局部特征。
